# Global Assets

### What goes here?
Files that are **shared** across multiple different parts of the game.
* **Textures:** Generic patterns (like `space.png`), tilesets, or UI borders.
* **Audio:** Common sound effects (button clicks) and music tracks.
* **Fonts:** `.ttf` or `.otf` files used for your UI text.

### Why?
If an asset is used by more than one "Feature," it belongs here. By centralizing shared assets, you avoid duplicating files and make it easier to swap out a texture or font project-wide in one click.
